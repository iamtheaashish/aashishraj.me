## Customization

### Update Content

1. Open any HTML file in a text editor
2. Replace placeholder content with your own
3. Update links, descriptions, and metadata

### Modify Styling

- Edit `styles.css` to change colors, spacing, or layout
- Current color scheme uses CSS variables in `:root`
- Change font by updating the Google Fonts import URL

### Add Functionality

- Edit `script.js` to add interactive features
- Current features: fade-in animations, keyboard navigation
